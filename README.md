# PalindromeCheckerApp
**UC2 – Check Palindrome Using Scanner**

* The objective of UC2 is to check whether a user-entered word is a palindrome.
* In this use case, the program starts from the main() method.
* The system asks the user to enter a word.
* The input is taken using the Scanner class.
* The entered word is stored in a String variable.
* The program compares characters from the beginning and end of the word.
* A loop is used to check only till half of the string length.
* If all characters match, the word is a palindrome.
* If any character does not match, it is not a palindrome.
* The result is displayed on the console.
* This use case introduces user input handling and comparison logic.
* The main class used is PalindromeCheckerApp.
* This use case moves from hardcoded logic to dynamic input checking.
