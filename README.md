# PalindromeCheckerApp
**UC8 - Linked List Based Palindrome Checker**

* The objective of UC8 is to check whether a string is a palindrome using a LinkedList.
* The program starts from the main() method.
* The user enters a word using the Scanner class.
* Each character of the string is stored in a LinkedList.
* LinkedList allows operations from both beginning and end.
* The program removes the first and last characters and compares them.
* The comparison continues until only one or zero elements remain.
* If all characters match, the string is a palindrome.
* If any characters do not match, it is not a palindrome.
* This use case demonstrates double-ended operations in LinkedList.
* The main class used is PalindromeCheckerApp.
