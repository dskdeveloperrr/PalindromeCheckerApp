# PalindromeCheckerApp
**UC1 - Application Entry & Welcome Message**

* The objective of this project is to create a simple Java application to check whether a word or number is a palindrome.
* The project is developed step-by-step using use cases.
* First, the application starts from the main() method.
* When the program runs, it displays a welcome message.
* It also shows the application version.
* Then it confirms that the system is initialized successfully.
* This is done in Use Case 1 (UC1).
* UC1 is only used to start the system and establish a proper startup flow.
* In UC1, no user input is taken.
* In UC1, no palindrome checking is performed.
* The main class used in this project is PalindromeCheckerApp.
* This class acts as the entry point of the application.
* Future use cases will take user input, process it, check for palindrome, and display the result.
